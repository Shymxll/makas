# Makas Frontend

Next.js frontend for Makas audio processing tool.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Technologies

- Next.js 16
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- wavesurfer.js
- lucide-react
